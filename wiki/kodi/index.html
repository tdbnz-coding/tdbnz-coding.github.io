How to Install and Use Kodi

Your ultimate guide to getting started with Kodi, the powerful media player.

Installing Kodi

Kodi is available on multiple platforms, including Windows, macOS, Android, iOS, and Linux. Follow these steps to install it on your preferred device:

Step 1: Visit the Official Kodi Website

Go to the official Kodi website: https://kodi.tv/download.

Step 2: Choose Your Operating System

Click on the icon for your operating system (Windows, macOS, Android, etc.).

Step 3: Download and Install Kodi

Download the installer for your device. Run the installer and follow the on-screen instructions to complete the installation.

Basic Setup

Step 1: Launch Kodi

After installation, open Kodi. You’ll see the default home screen.

Step 2: Add Media Sources

Go to Settings (gear icon) → File Manager → Add Source. Enter the location of your media files or an external source.

Step 3: Customize Appearance (Optional)

Navigate to Settings → Interface → Skin to change the look and feel of Kodi.

Using Kodi

1. Playing Media

Select the type of media you want to play (Movies, TV Shows, Music, etc.) from the home screen. Navigate to your added media sources and choose a file to play.

2. Installing Add-ons

Go to Add-ons from the home screen. Click the Download option and browse the official Kodi repository for add-ons.

3. Managing Settings

Use the Settings menu to adjust video playback, audio output, network settings, and more.

Installing Repositories and Add-ons

Kodi supports third-party repositories and add-ons to enhance functionality. A highly recommended source for NZ users is the Sly Guy Repository - https://slyguy.uk/, which offers Freeview NZ, TVNZ+, and more.

Step 1: Enable Unknown Sources

To install third-party add-ons, go to Settings → System → Add-ons, and toggle Unknown Sources to On. Confirm the warning message.

Step 2: Add the Repository Source

Go to Settings → File Manager → Add Source. Enter the following URL:

  • https://slyguy.uk

Name the source SlyGuy and click OK.

Step 3: Install the Repository

Go back to the Kodi home screen and select Add-ons. Click the open box icon and choose Install from ZIP file. Navigate to the SlyGuy source and select the ZIP file to install the repository.

Step 4: Install Add-ons

After installing the repository, choose Install from Repository, select the Sly Guy Repository, and browse available add-ons like Freeview NZ and TVNZ+.

Important Notes:
  • Only install add-ons from trusted sources to avoid malware.
  • Keep your Kodi updated to ensure compatibility with repositories and add-ons.
