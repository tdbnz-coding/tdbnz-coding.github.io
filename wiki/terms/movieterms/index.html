Your ultimate Movie Guide

High-Quality Sources

Blu-ray Disc/Remux

The original high-definition movie file from a physical Blu-ray disc. Remux removes extras for reduced size.

WebDL

Digital file directly from a streaming service, offering excellent quality without intermediate compression.

Blu-ray Encode

A compressed version of the Blu-ray source to reduce file size while maintaining high quality.

Pre-Blu-ray/Theater-Sourced Copies

HDTC (Telecine)

Created using a telecine machine, with potential artifacts from the film reel.

HDTS (Telesync)

Recorded in a theater with external audio for improved sound. Visuals often distorted.

HQ-CAM

A recording made in a theater, often with poor lighting and audience sounds.

Resolutions and Streaming Speeds

SD (Standard Definition)

Resolution: 480p (640x480). Suitable for smaller screens. File sizes: 700MB–1.5GB/hour.

Streaming Speed: 3 Mbps or higher.

HD (High Definition)

Resolution: 720p/1080p. Suitable for most modern TVs. File sizes: 1.5GB–5GB/hour.

Streaming Speed: 5–8 Mbps or higher.

HQ (High Quality)

High-bitrate 1080p content. File sizes: 4GB–10GB/hour depending on compression.

Streaming Speed: Minimum 10 Mbps.

4K (Ultra HD)

Resolution: 2160p. Best for large screens. File sizes: 7GB–20GB/hour.

Streaming Speed: 25 Mbps or higher.
